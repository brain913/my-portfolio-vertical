# My Portfolio Vertical

This project is a personal portfolio built using React. It showcases various projects and skills in a visually appealing manner, utilizing modern web technologies and design principles.

## Project Structure

- **public/**: Contains static assets such as models and gallery images.
  - **gallery/**: Directory for gallery images.
  - **models/**: Directory for 3D models.
  
- **src/**: The source code for the application.
  - **components/**: Contains reusable React components.
    - **sections/**: Section components for the portfolio.
    - **ui/**: Reusable UI elements like buttons and cards.
  - **data/**: Contains data related to the portfolio.
  - **theme/**: Centralized theme tokens for consistent styling.
  - **animations.js**: Shared motion utilities for animations.
  - **index.css**: Global CSS styles.
  - **main.jsx**: Entry point of the application.
  - **portfolio.jsx**: Main composition file for layout and sections.

## Getting Started

To get started with the project, follow these steps:

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd my-portfolio-vertical
   ```

3. Install dependencies:
   ```
   npm install
   ```

4. Start the development server:
   ```
   npm run dev
   ```

5. Open your browser and go to `http://localhost:3000` to view the portfolio.

## Scripts

- **dev**: Starts the development server.
- **lint**: Lints the codebase.
- **build**: Builds the project for production.
- **preview**: Previews the production build.
